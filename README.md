## Packages

- React
- Axios

  ## Running the project

Clone this repository :

```

git clone https://github.com/bangtam1994/leboncoin-front.git

cd leboncoin-front

```

To install packages :

```
npm install
```

or

```
yarn add
```

When installation is complete, run the project with:

```
yarn start or npm start
```

## Deployment
